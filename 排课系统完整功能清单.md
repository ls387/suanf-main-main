### 2.3 遗传操作之交叉（Crossover）

**位置**：`genetic_algorithm.py` L1249-1262

#### 2.3.1 单点交叉机制

```python
def crossover(parent1, parent2) -> (child1, child2):

    if random() > crossover_rate (default 0.85):
        return parent1[:], parent2[:]  # 85%概率执行交叉

    # 单点交叉：随机选择分割点
    point = randint(1, min(len(parent1), len(parent2)) - 1)

    child1 = parent1[:point] + parent2[point:]    # 前段来自P1，后段来自P2
    child2 = parent2[:point] + parent1[point:]    # 前段来自P2，后段来自P1

    return child1, child2
```

#### 2.3.2 生物学解释

- **高交叉率0.85**（高于传统0.7-0.8）
  - 目的：增加种群多样性
  - 好处：避免早期收敛陷阱
  - 代价：可能增加计算量（但相对可控）

#### 2.3.3 特点

- ✅ 保持较好的架构（保留连续的时间安排块）
- ✅ 引入多样性（两条不同的排课方案混合）
- ✅ 计算复杂度 O(n)，效率高
